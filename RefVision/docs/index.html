The read

Evidence from pose, a marker you control, a call you can defend.

RefVision doesn't overrule the referee. It measures the challenge from two-person body pose, hands the point of contact to the official to confirm, and shows its own read next to the on-field decision — flagging anything close for review.

Contact moment, found

Two-person MediaPipe pose locates the frame where the challenge lands — gated to upright players so it reports the strike, not the fall that follows.

You place the marker

Click the exact point of contact on the frame. The engine's estimate is only a starting position — reliable even on fused scrambles a model can't resolve alone.

±n replay, paused at contact

A windowed replay around the marked frame. The marker pops up at the instant of contact and the frame holds for two seconds, then plays on.

Graded, not guessed

Speed scaled to each player's own shoulder-to-ankle height in frame; boot elevation and studs read across the approach, not one frame; contact height factored in.

Holding & aerial screening

Hand-on-torso holds and aerial duels — including an arm or elbow to the head — are flagged for review across every player on the pitch, goalkeepers included.

Decision vs the signal

Record the on-field call and RefVision lays its read beside it — matching or diverging — so a disagreement is visible and sent to review, never buried.

How it works

Five steps from footage to a defensible call.

A single pass of pose, then measurement, then a human in the loop. Ordered, because each step depends on the one before it.

01

Extract pose

MediaPipe runs on every sampled frame of the clip. Players are separated by shirt colour so the two in the challenge can be tracked apart.

02

Locate contact

The engine picks the earliest in-contact frame while both players are still upright — the strike — rather than the tangle they collapse into afterward.

03

Measure the challenge

Closing speed, boot elevation, studs exposure and contact height are read across the approach window and combined into a leg-challenge severity; holds and aerial duels are screened in parallel.

04

Confirm the marker

In the Crop & mark tab the official scrubs to the contact frame, clicks the true point of contact, and sets the ±n replay window. Confirm locks it.

05

Decide

Severity becomes a call — no foul, foul, caution, or sending-off — shown against the referee's signal, with anything near a threshold marked for review.

Run it

Activate the Streamlit console.

Python 3.11, no GPU. The pose model downloads once on the first run, so the first launch needs a connection.

Clone the repo (or unzip the project) and run the commands on the right from its folder.

On Windows, launch with python -m streamlit run app.py — the -m form avoids PATH issues with Microsoft Store Python.

streamlit-image-coordinates is optional; with it you place the contact point by clicking the frame, without it you get sliders.

The console opens at http://localhost:8501.

bash — FIFA 
# 1 · get the project
git clone https://github.com/wenjun78/FIFA.git
cd FIFA

# 2 · create & activate a virtual environment
python -m venv .venv
# Windows (PowerShell)
.venv\Scripts\Activate.ps1
# macOS / Linux
source .venv/bin/activate

# 3 · install dependencies
python -m pip install -r requirements.txt

# 4 · optional: click-to-place the contact point
python -m pip install streamlit-image-coordinates

# 5 · launch the console  →  http://localhost:8501
python -m streamlit run app.py
The console

Two tabs: mark it, then decide.

The whole review is one loop — set the contact point on the left, read the call on the right.

Tab 01

Crop & mark

Everything needed to fix the moment of contact, in one place.

  • Scrub to the contact frame (scrollbar or type the time)
  • Click the frame to place the contact marker
  • Set the ± replay window around it
  • Confirm — jumps you to the decision
Tab 02

Decision

The confirmed marker, the measurements, and the call.

  • ±n replay that pauses ~2 s on the contact frame
  • Pose measurements — speed, boot, studs, height
  • The pose call, with confidence and review flag
  • Laid beside the referee's on-field signal

RefVision is decision-support, not an automated referee. It measures, marks and compares — the official makes the call. Most reliable on cleanly-framed or slow-motion footage.
